# Performance Tooling

Available scripts:
- `k6_smoke.js`: lightweight API smoke performance check
- `k6_ramp_spike_soak.js`: full profile load test script with SLO thresholds
- `run_load_profiles.ps1`: executes ramp, spike, and soak profiles with summary export
- `failure_mode_checks.ps1`: sanity checks for failure-mode behavior

See `LOAD_TEST_PLAN.md` for execution and acceptance criteria.
