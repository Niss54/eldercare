# Production Deployment Checklist

- [ ] Environment variables sourced from secret manager
- [ ] Database migration plan reviewed and approved
- [ ] Blue/green or canary rollout strategy selected
- [ ] On-call owner assigned for deployment window
- [ ] Rollback plan verified
- [ ] Health checks and readiness probes validated
- [ ] Alerts and dashboards live
- [ ] Post-deploy smoke tests pass
- [ ] Backup snapshot captured pre-deploy
- [ ] Deployment signoff recorded
